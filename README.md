# sudoku_solver

App where you can solve sudoku puzzles by yourself, check solutions or set sudoku puzzles for computers to solve.

The solver part convert sudoku puzzle to a [exact cover problem](https://en.wikipedia.org/wiki/Exact_cover#Sudoku) which is then solved by Algorithm X implemented using [Dancing Links](https://en.wikipedia.org/wiki/Dancing_Links).

## Installation

From project folder run:
`make install`

## Usage
From project folder run:
`python3 gui/main.py`
